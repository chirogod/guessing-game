Juego de adivinar el numero.
URL de la pagina del proyecto: https://roadmap.sh/projects/number-guessing-game

# Requisitos

- El juego comienza y debe mostrar un mensaje de bienvenida con las reglas.
- La computadora elige un numero al azar entre 1 y 100.
- Se selecciona la dificultad lo que determinara el numero de intentos para adivinarlo.
- Usuario ingresa su prediccion y se determina si adivino o no.

# Instalacion
Para usar la aplicacion, seguir los siuientes pasos:
1. Clonar el repositorio:

        git clone https://github.com/chirogod/guessing-game.git 

2. Navegar a la raiz del proyecto

		cd guessing-game

3. Correr la aplicacion

		dotnet run

4. Comenzar a jugar!!!
